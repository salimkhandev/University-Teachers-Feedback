## PHASE 3 — Auth System

### 3A. JWT utilities (`src/utils/jwt.ts`)
- `generateAccessToken(userId, role)` — expires in 15 minutes
- `generateRefreshToken(userId)` — expires in 7 days
- `verifyAccessToken(token)` — returns payload or throws
- `verifyRefreshToken(token)` — returns payload or throws

### 3B. Auth middleware (`src/middleware/auth.ts`)
- Extract Bearer token from `Authorization` header
- Verify with `verifyAccessToken`
- Attach `req.user = { id, role }` to request
- Return 401 if missing or invalid

### 3C. RBAC middleware (`src/middleware/rbac.ts`)
```typescript
export const requireRole = (...roles: string[]) =>
  (req, res, next) => {
    if (!roles.includes(req.user.role)) return res.status(403).json({ error: 'Forbidden' });
    next();
  };
```

### 3D. Auth routes (`src/routes/auth.ts`)

**POST /api/auth/login**
1. Find user by username
2. Compare password with bcrypt
3. Return `{ accessToken, refreshToken, user: { id, name, role } }`

**POST /api/auth/refresh-token**
1. Verify refresh token
2. Return new access token

**Verify Phase 3:** Login with a seeded admin user returns two tokens.

```typescript
const assignments = await TeacherAssignment.find({ teacherId });
const assignmentIds = assignments.map(a => a._id);
const currentCount = await Feedback.countDocuments({ assignmentId: { $in: assignmentIds } });

const cached = await AISummary.findOne({ teacherId });
if (cached && cached.feedbackCount === currentCount) {
  return res.json({ summary: cached.summaryText, cached: true });
}

// Call Gemini
const comments = await Feedback.find({ assignmentId: { $in: assignmentIds } })
  .select('comment rating');
const summary = await callGemini(comments);

await AISummary.findOneAndUpdate(

return res.json({ summary, cached: false });
```

## PHASE 9 — Gemini Service

File: `src/services/gemini.ts`

```typescript
import { GoogleGenerativeAI } from '@google/generative-ai';

const genAI = new GoogleGenerativeAI(process.env.GEMINI_API_KEY!);
const model = genAI.getGenerativeModel({ model: 'gemini-pro' });

export async function generateTeacherSummary(comments: string[], ratings: number[]) {
  const prompt = `
You are analyzing student feedback for a university teacher.
Average rating: ${average(ratings)}/10
Total responses: ${ratings.length}

Student comments:
${comments.map((c, i) => `${i+1}. "${c}"`).join('\n')}

Write a professional 3-paragraph summary:
1. Overall performance and student sentiment
2. Key strengths mentioned
3. Areas for improvement

Be specific. Do not invent details not in the comments.
  `;
  const result = await model.generateContent(prompt);
  return result.response.text();
}

export async function chatWithTeacher(messages: {role: string, content: string}[], context: string) {
  const chat = model.startChat({
    history: messages.map(m => ({ role: m.role, parts: [{ text: m.content }] }))
  });
  const result = await chat.sendMessage(context);
  return result.response.text();
}
```

**Verify Phase 9:** Summary generates without error. Response is coherent text.
